## 📂 开放技能（Open Skills）

### [`toutiao-fabu`](./toutiao-fabu/) — 自动化发布 Loop 引擎

无人值守的内容发布系统：库存模型（产-消解耦）+ 调度器 + 五大护栏 + 失败熔断 + 登录态双验证。

> 从「今日头条微头条」实战沉淀的工程范式，适用于任何需要定时定量发布的自动化场景（公众号、小红书、知乎、邮件推送等）。

**核心可迁移智慧**：
- 📦 库存模型：producer 写稿 / consumer 发稿，目录解耦
- 🛡️ 五大护栏：窗口 / 失败计数 / 重复检测 / 间隔 / 上限
- ⚡ 失败熔断：三档分级（登录态 / 连续失败 / 单条失败）
- 🔐 登录态双验证：cookie + 页面 DOM
- ⏰ 调度算法：behind 负数修正 + 尾部兜底

最初设计场景：今日头条微头条全自动发布（v1.5.2）。
