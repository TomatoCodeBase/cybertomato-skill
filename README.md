### [`toutiao-cadence`](./toutiao-cadence/) — 今日头条自动化发布引擎

写好的内容丢给它，它自己按时间表一条条发出去——白天发、半夜也发，你该睡觉睡觉。每发一条通知你一条，发完就停。登录过期或者连续发失败，它会自己停下来喊你处理，不会傻乎乎一直刷错把账号搞封。同一套方法换个平台（公众号、小红书、知乎）也能跑。

最初设计场景：今日头条微头条全自动发布（v1.5.2）。
